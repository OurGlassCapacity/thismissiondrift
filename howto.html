Using the Dropdown Tool in Processing

What is "Processing"?
in Processing

Run the program by pressing the 'Run' button. [Cmd-R]

Select the secondary window. Begin typing the body of text.

Once encountering a dropdown marker, copy a formatted list of terms. [Select > Cmd-V]

       *Be sure to exclude any ending punctuation on the last term of a list, such as a comma.

Return to the dropdown text window. Select it so that it is active.

Paste the terms into the window using [TAB]

Continue with written text. Repeat the process when encountering another dropdown marker.

When concluding, in the case where words end a body of text- rather than a dropdown term, use the shortcut [`] This will add the last written text to the formatter.

       *This shortcut key is located congruently with the [~] 'tilde' key.
